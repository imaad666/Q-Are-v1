# Q[Are] - v1 -  Personal QR Login Generator for your session QRs

A custom QR code generator app for training login codes, built with modern Android technologies.

## Features

### 🎯 Core Functionality
- **One-time Setup**: Upload your old QR code to extract your details
- **Daily QR Generation**: Automatically generates fresh QR codes for every day
- **Offline Operation**: Works completely offline - no internet required
- **Smart Validation**: Ensures QR codes match the correct login format

### 🎨 Customization (Qewie-inspired)
- **Color Selection**: Choose from 8 different QR code colors
- **Corner Radius**: Adjust rounded corners for a modern look
- **Initials Overlay**: Add your 2-letter initials to the QR code
- **Real-time Preview**: See changes instantly as you customize

### 📱 User Experience
- **Welcome Screen**: Clean app introduction
- **Intuitive Setup**: Simple one-time QR upload process
- **Settings Panel**: Easy profile management and reset options
- **Modern UI**: Built with Material Design 3 and Jetpack Compose

## How It Works


1. **Setup Process**: Upload an old QR code image
2. **Validation**: App extracts your QR details
4. **Daily Use**: App automatically the day's fresh QR that can be customized
5. **Customization**: Personalize your QR with colors, corners, and initials

## Technical Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose with Material Design 3
- **QR Processing**: ZXing library for encoding/decoding
- **Navigation**: Navigation Compose
- **Data Storage**: SharedPreferences for offline persistence
- **Image Loading**: Coil for efficient image handling


## Version

- **Current Version**: 1.0
- **Min SDK**: 24 (Android 7.0+)
- **Target SDK**: 36
